Project Overview:

This project implements a BCD (Binary Coded Decimal) Adder using Verilog HDL.
It adds two 4-bit BCD numbers and produces a valid BCD output with carry correction logic.

Working Principle:
A BCD adder works in two steps:
Add two 4-bit binary numbers using a ripple carry adder
If the sum > 9 (1001), add 0110 (6) to correct it into valid BCD code
