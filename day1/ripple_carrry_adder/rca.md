Project Overview:

This project implements a 4-bit Ripple Carry Adder (RCA) using Verilog HDL.
It performs binary addition of two 4-bit numbers using cascaded full adders.

 Working Principle:
 
A ripple carry adder is built using multiple full adders connected in series.
Each full adder adds:
A bit from input A
A bit from input B
Carry from previous stage
The carry “ripples” from LSB to MSB
