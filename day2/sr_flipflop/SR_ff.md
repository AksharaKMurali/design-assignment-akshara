SR Flip-Flop

 Overview:

An SR Flip-Flop stores 1-bit memory using Set and Reset inputs.

Truth Table:

S=1, R=0 → Set (Q=1)

S=0, R=1 → Reset (Q=0)

S=0, R=0 → No change

S=1, R=1 → Invalid
