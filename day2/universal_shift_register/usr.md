Universal Shift Register 

 Overview:

A Universal Shift Register can:

Hold data

Shift left

Shift right

Parallel load

 Modes
| S1 | S0 | Operation     |
| -- | -- | ------------- |
| 0  | 0  | Hold          |
| 0  | 1  | Shift Right   |
| 1  | 0  | Shift Left    |
| 1  | 1  | Parallel Load |
